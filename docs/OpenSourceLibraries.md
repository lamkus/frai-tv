# Open‑Source‑Bibliotheken und -Projekte

In diesem Dokument sind externe Bibliotheken und Open‑Source‑Projekte
aufgeführt, die für den Aufbau einer Streaming‑Mediathek genutzt oder
studiert werden können. Sie dienen als Referenz, Inspiration oder
Direkteinsatz. Bitte prüft vor der Nutzung die jeweiligen Lizenzen.

| Name | Beschreibung | Link |
|-----|-------------|------|
| **Streama** | Selbst gehosteter Streaming‑Server. Bietet ein Netflix‑ähnliches UI, "Continue Watching", Fortschrittsanzeige, Episode‑Browser und Admin‑Panel. | **Nextflix** | Open‑Source‑Netflix‑Clone mit moderner Architektur. Verwendet Next.js 13, TypeScript, Server Actions, mehrere Profile, Suchfunktion und unendliches Scrollen【832452663600043†screenshot】. Ideal als Vorlage für Frontend‑Struktur und Nutzerprofile. | <https://github.com/Apestein/nextflix> |
| **Codesandbox Netflix Clone** | Einfacher Netflix‑Clone, implementiert mit HTML, CSS und JavaScript. Hilft beim Verständnis von Layouts und Grundstrukturen. | <https://codesandbox.io/s/netflix-clone> |
| **Angular Netflix Clone** | Umsetzung eines Netflix‑Klon in Angular. Zeigt, wie man horizontale Scroller und filmähnliche Kachelansichten mit Angular realisiert. | <https://github.com/sadmann7/netflix-web> |
| **Embed Plus for YouTube** | WordPress‑Plugin zum Einbetten kompletter YouTube‑Kanäle, Playlists und Live‑Streams. Nützlich für Low‑Code‑Lösungen; unterstützt eine paginierte Galerie und "facade mode" zum Lazy‑Loading. | <https://wordpress.org/plugins/youtube-embed-plus/> |
| **Video.js** | JavaScript‑Bibliothek für HTML5‑Videos. Falls in Zukunft andere Videoquellen (MP4, HLS) eingebunden werden, eignet sich Video.js als Player. | <https://github.com/videojs/video.js> |
| **hls.js** | Clientseitige Bibliothek zum Abspielen von HLS‑Streams im Browser. Relevant, wenn Livestreams ausserhalb von YouTube genutzt werden. | <https://github.com/video-dev/hls.js> |
| **Tailwind CSS** | Utility‑First‑CSS‑Framework, das in diesem Projekt für flexible Designs und Theme‑Wechsel verwendet wird. | <https://tailwindcss.com/> |
| **Prisma / TypeORM** | ORMs für Node.js zur einfachen Modellierung und Verwaltung relationaler Datenbanken (PostgreSQL). | <https://www.prisma.io/> / <https://typeorm.io/> |
